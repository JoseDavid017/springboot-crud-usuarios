# API CRUD de Usuarios

Este proyecto es una API REST desarrollada con Spring Boot.

## 🚀 Funcionalidades

- Crear usuario
- Listar usuarios
- Actualizar usuario
- Eliminar usuario
- Validaciones
- Manejo de errores

## 🛠️ Tecnologías

- Java
- Spring Boot
- MySQL
- JPA (Hibernate)

## 📡 Endpoints

POST /usuarios  
GET /usuarios  
PUT /usuarios/{id}  
DELETE /usuarios/{id}
